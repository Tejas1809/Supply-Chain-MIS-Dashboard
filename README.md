# 📦 Supply Chain MIS Dashboard

This project presents a dynamic **Supply Chain Management Information System (MIS) Dashboard** built in **Microsoft Excel**. 
It enables end-to-end visibility into key supply chain metrics such as inventory availability, supplier performance, logistics costs, and reorder status.

---

## 🔍 Project Objectives
- Monitor real-time inventory levels and product availability.
- Track revenue, order quantities, and product movement.
- Identify reorder needs and automate alerts for stock shortages.
- Analyze supplier performance via defect rates and lead times.
- Visualize shipping costs and route performance.

---

## 📁 Features

✅ **KPI Metrics**
- Total Revenue
- Total Products Sold
- Average Defect Rate
- Reorder Alerts Count

📊 **Visual Dashboards**
- Revenue by Product Type
- Stock vs Order Quantities
- Supplier Defect Distribution
- Shipping Cost by Carrier
- Route Cost Analysis

🧩 **Interactive Filters**
- Product Type
- Supplier Name
- Location

📎 **Tools Used**
- Microsoft Excel (Pivot Tables, Charts, Slicers)
- Conditional Formatting
- Calculated Fields
